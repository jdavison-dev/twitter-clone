# MERN Stack Project

- Tech Stack: React.js, MongoDB, Node.js, Express, Tailwind
- React Query for Data Fetching, Caching etc.
- Suggested Users to Follow
- Create Posts
- Like Posts
- Comment on Posts
- Repost
- Delete Posts
- Edit your profile image, cover image, and bio
- View following posts

### Build the app

```shell
npm run build
```

### Start the app

```shell
npm start
```
